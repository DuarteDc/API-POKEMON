# Modo desarrollo 
1. Ejecutar
```
yarn install
```
2. Instalar Nest CLI
```
npm i -g @nestjs/cli
```
3. Iniciar la base de datos
```
docker-compose up -d
```

4. Reconstruir la base de datos
```
http://localhost:3000/api/v2/seed
```
